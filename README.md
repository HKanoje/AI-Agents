## Project Setup (In Command Promt)

### Create a Virtual `.venv`
```bash
python -m venv .venev
```
### Activate the `.venv`
```
.venv\Scripts\activate
```
### Install `requirements.txt`
```
pip install -r requirements.txt
```

## How to Run Ollama (Keep it Running whole time)
In another Command Promt
```
ollama serve
```
